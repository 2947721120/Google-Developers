## 远程调试协议 1.0

协议被份为几个部分（如：DOM, Debugger, Network 等），每个部分都定义了一些支持的命令和事件。命令和事件都是固定的序列化 JSON 对象。你可以使用在线调试，也可以使用调试器扩展 API。

- [console](https://developer.chrome.com/devtools/docs/protocol/1.0/console)
- [DOM Debugger](https://developer.chrome.com/devtools/docs/protocol/1.0/domdebugger)
- [Page](https://developer.chrome.com/devtools/docs/protocol/1.0/page)
- [Debugger](https://developer.chrome.com/devtools/docs/protocol/1.0/debugger)
- [Runtime](https://developer.chrome.com/devtools/docs/protocol/1.0/runtime)
- [DOM](https://developer.chrome.com/devtools/docs/protocol/1.0/dom)
- [Network](https://developer.chrome.com/devtools/docs/protocol/1.0/network)
- [Timeline](https://developer.chrome.com/devtools/docs/protocol/1.0/timeline)
